# Ocado Smart Platform Presentation

This directory contains an HTML presentation about the Ocado Smart Platform.

## How to View

1.  **Open `presentation.html`** in your web browser (Chrome, Safari, Firefox, etc.).
2.  You will see the slides laid out vertically.

## How to Convert to PDF (for Google Slides)

Since this is a web-based presentation, you can print it to PDF to import it into other tools or share it.

1.  Open `presentation.html` in Chrome.
2.  Press `Cmd + P` (Mac) or `Ctrl + P` (Windows) to print.
3.  Set "Destination" to "Save as PDF".
4.  Ensure "Background graphics" is checked in the "More settings" section.
5.  Save the PDF.

## How to use in Google Slides

Google Slides does not directly import HTML. However, you can:
1.  **Present the HTML directly**: Just use the browser as your presentation tool.
2.  **Use the PDF**: Upload the generated PDF to Google Drive and present from there, or convert the PDF pages to images and place them on slides.
